# ShopSphere – E-Commerce Management System

## Project Description

ShopSphere is a full-stack e-commerce management system developed using modern web technologies. The application allows users to register, log in, manage products, and place orders through a responsive and user-friendly interface. It demonstrates a complete integration between a React frontend and a Spring Boot backend using RESTful APIs.

The backend is built with Java and Spring Boot, providing secure and scalable APIs for handling user management, product management, and order processing. PostgreSQL is used as the database to store application data, ensuring efficient and reliable data handling. Postman was used to test and validate REST APIs during development.

The frontend is developed using React, creating a dynamic single-page application where users can interact with the system seamlessly. The project follows a layered architecture including Controller, Service, Repository, and Entity layers to maintain clean and maintainable code.

This project demonstrates full-stack development skills including frontend development, backend API creation, database integration, and API testing.
